How to run:
1. run in terminal of root folder: "npm install"
2. run in terminal of root folder: "npm install --save-dev cross-env"
3. Create ".env" file and put these inside:
- MONGODB_URI=mongodb+srv://<username>:<password>@cluster0.w8dipua.mongodb.net/<database>?retryWrites=true&w=majority&appName=Cluster0
- SESSION_SECRET=yourStrongSecretKey
- PROMOTE_SECRET=themanwhosoldtheworld
- PORT=5000
4. Connect the database through MONGODB_URI.
